# 示例输入

你可以这样给 Codex 提需求：

```text
这是 EP01 的素材目录：episodes/EP01_我的第一条视频日记

主题：
今天想讲我为什么开始做视频日记。不是为了做精致内容，而是想记录每天真实推进项目的过程。

素材：
- raw/talking-head.mp4 是口播原片
- raw/talking-head.srt 是原始字幕
- assets/project-screenshot.png 是项目截图
- assets/chat-recording.mp4 是工作录屏

要求：
- 保持真实口播感
- 需要中英双语字幕
- 截图出现时要放大，别做小卡片
- 不要加二维码

请先检查素材和剪辑规划，我确认后再开始渲染。
```
