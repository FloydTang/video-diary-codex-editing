# 单期目录说明

把这一整个目录复制到 `episodes/EPXX_标题/`，或使用：

```bash
bash scripts/new_episode.sh EP01 标题
```

## 文件说明

- `raw/`：放原始口播视频、原始 SRT、原始转写稿。
- `assets/`：放截图、录屏、网页导出、图片、证明素材。
- `粗剪母版/`：如果原片不干净，先放口播清洗母版、对齐 SRT 和删减表。
- `口播稿.md`：如果还没录制，先在这里整理脚本；如果已录制，放主题和转写摘要。
- `素材清单.md`：列出素材用途和隐私风险。
- `剪辑规划.md`：Codex 先补规划，再进入制作。

## 给 Codex 的建议提示

```text
请按 AGENTS.md 检查本期素材，先输出剪辑规划和缺失素材判断。如果原片不干净，先做粗剪母版给我确认；确认后再进入正式剪辑落地。
```
