# 隐私与公开前检查

公开视频最容易泄露的信息，往往藏在截图和录屏里。

## 必查内容

发布前逐帧检查这些信息：

- 手机号
- 邮箱
- 微信号、飞书号、WhatsApp、Telegram
- 客户姓名、公司名、头像
- 订单号、合同号、发票号
- 地址、物流单号
- 后台金额、利润、报价
- API key、token、cookie、二维码
- 浏览器地址栏里的私密链接
- 文档权限链接
- 内部群聊成员、客户聊天原文
- 未授权的人脸

## 脱敏方式

优先级：

1. 重新裁切，只保留有用区域。
2. 用模糊遮住敏感区域。
3. 用纯色块遮住。
4. 换成示意图或重做一张无隐私截图。

不要只靠“画面很小应该看不清”。高清视频和暂停截图会暴露细节。

## Codex 执行要求

如果发现敏感信息：

- 不要继续渲染最终版。
- 在 `剪辑规划.md` 标出风险素材和时间点。
- 给出最小脱敏方案。

如果用户坚持使用敏感素材，最终回复必须明确提醒风险。

## 公开仓库检查

发布这个模板仓库前，确认没有：

- 真实素材
- 私人路径
- 真实项目名
- 真实客户名
- 成片
- 大体积视频
- `.DS_Store`
- 缓存和中间渲染
- 个人账号信息
